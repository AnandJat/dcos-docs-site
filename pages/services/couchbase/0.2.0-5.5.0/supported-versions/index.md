# Supported Versions

## Package Versioning Scheme

- Couchbase: 5.5.0
- Couchbase Sync Gateway: 2.0.0
- DC/OS: 1.10 and 1.11

Packages are versioned with an a.b.c-x.y.z format, where a.b.c is the version of the DC/OS integration and x.y.z indicates the version of Couchbase. For example, 0.2.0-5.5.0 indicates version 0.2.0 of the DC/OS integration and version 5.5.0 of Couchbase Server.

## Version Policy

The DC/OS Couchbase Service is engineered and tested to work with a specific release of Couchbase Server and Couchbase Sync Gateway.
