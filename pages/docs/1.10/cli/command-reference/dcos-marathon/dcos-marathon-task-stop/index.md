# Description
Stop a task.

# Usage

```bash
dcos marathon task stop <task-id> [OPTION]
```

# Options

| Name, shorthand | Default | Description |
navigationTitle:  dcos marathon task stop
|---------|-------------|-------------|
| `--wipe`   |             | Wipe persistent data. |

# Positional arguments

navigationTitle:  dcos marathon task stop
|---------|-------------|-------------|
