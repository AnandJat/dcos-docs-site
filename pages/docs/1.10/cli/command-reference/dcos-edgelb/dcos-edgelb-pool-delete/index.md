# Description
Delete and uninstall an Edge-LB pool.

# Usage

```bash
dcos edgelb [<flags>] pool delete <name>
```

# Positional arguments

| Name, shorthand | Default | Description |
navigationTitle:  dcos edgelb pool delete
|---------|-------------|-------------|
| `<name>`   |             |  Pool name. |
