# Description
Shutdown a service.

# Usage

```bash
dcos service shutdown <service-id> [OPTION]
```

# Options

None. 

# Positional arguments

| Name, shorthand | Default | Description |
navigationTitle:  dcos service shutdown
|---------|-------------|-------------|
| `<service-id>`   |             | The DC/OS service ID. |
